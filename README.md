# Simulateur de participation aux frais — Caen Falaise Planeurs

A single-page web tool for simulating flight cost participation at the [Caen Falaise Planeurs](https://cfp-planeur.fr) gliding club.

## Overview

This simulator helps pilots and club members estimate their share of flight costs based on various scenarios (solo, dual, tow, etc.). It runs entirely in the browser — no server, no dependencies, no build step.

## Usage

Just open `index.html` in any modern browser. No installation required.
